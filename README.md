## Streeetching font toooools 

This is an experiment to create tools that make use of the "stretching" abilities of the SOS font. 

The stretched letter is obtained by the interpolation of two glyphs.

## index.html
A stretching composer. The longer you press the more you stretch!

## sound.html
A stretching composer. The longer you press the more you stretch!

## Usage
You must run this on a web server. Quickest way to achieve this on macOS is to use the built in Python server.
